== Description ==
Enhance your website with a diverse collection of Gutenberg patterns and Elementor section. This plugin adds numerous pre-designed patterns accessible through the block inserter on any page or via Elementor editor. These patterns serve as excellent starting points for creating various types of pages, blog posts, or service offerings, streamlining your design process and saving valuable time.

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'Super Blank'
3. Activate the plugin from your Plugins page.

= From WordPress.org =

1. Download 'Super Blank'.
2. Upload the 'Super Blank' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate 'Super Blank' from your Plugins page.
4. Go to any page in editor mode and open the inserter.
5. Select Patterns -> All.
6. Now you can select and insert the new patterns to your website.
